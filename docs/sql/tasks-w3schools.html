Рівень 1. Базова вибірка

1. Вибрати всіх клієнтів з таблиці Customers.
Де подивитись приклад: W3Schools → SQL SELECT
2. Вибрати ідентифікатор, назву клієнта та країну (CustomerID, CustomerName, Country).
Де подивитись приклад: W3Schools → SQL SELECT
3. Вибрати назву продукту та ціну. Відсортувати за ціною від дешевих до дорогих.
Де подивитись приклад: W3Schools → SQL ORDER BY

Рівень 2. Фільтрація

4. Вибрати всі продукти, у яких ціна більша за 60.
Де подивитись приклад: W3Schools → SQL WHERE
5. Вибрати всіх постачальників з країни Germany.
Де подивитись приклад: W3Schools → SQL WHERE
6. Вибрати всіх клієнтів з міста London.
Де подивитись приклад: W3Schools → SQL WHERE
7. Вибрати працівників з іменем Robert.
Де подивитись приклад: W3Schools → SQL WHERE
8. Вибрати всі замовлення після 1997-01-01.
Де подивитись приклад: W3Schools → SQL WHERE
9. Вибрати продукти з ціною від 10 до 30.
Де подивитись приклад: W3Schools → SQL BETWEEN
10. Вибрати клієнтів, які не з країни USA.
Де подивитись приклад: W3Schools → SQL WHERE (NOT)
