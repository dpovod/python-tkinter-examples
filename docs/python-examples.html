Приклади простих програм мовою Python

Без функцій

1. Привітання

Користувач вводить ім'я. Програма відповідає, наприклад: "Привіт, Сергій". У відповіді програми має бути ім'я користувача.


name = input("Введіть своє ім'я: ")
print("Привіт,", name + "!")

2. Сума двох чисел

Користувач вводить два числа. Програма рахує їх суму і повертає користувачу.


a = int(input("Введіть перше число: "))
b = int(input("Введіть друге число: "))
suma = a + b
print("Сума чисел =", suma)

Ті самі приклади, але через функції

1. Привітання (через функцію)

Тепер код оформлено у вигляді функції greet(), яку можна викликати в будь-якому місці програми.


def greet():
    name = input("Введіть своє ім'я: ")
    print("Привіт,", name + "!")

# Виклик функції
greet()

2. Сума двох чисел (через функцію)

Тут створено функцію sum_two_numbers(), яка запитує два числа, рахує їх суму і показує результат.


def sum_two_numbers():
    a = int(input("Введіть перше число: "))
    b = int(input("Введіть друге число: "))
    suma = a + b
    print("Сума чисел =", suma)

# Виклик функції
sum_two_numbers()
