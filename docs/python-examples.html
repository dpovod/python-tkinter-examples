Приклади простих програм мовою Python

Лінійні алгоритми (структура слідування)

Без функцій

1. Привітання

Користувач вводить ім'я. Програма відповідає, наприклад: "Привіт, Сергій". У відповіді програми має бути ім'я користувача. 


name = input("Введіть своє ім'я: ")
print("Привіт,", name + "!")

2. Сума двох чисел

Користувач вводить два числа. Програма рахує їх суму і повертає користувачу. 


a = int(input("Введіть перше число: "))
b = int(input("Введіть друге число: "))
suma = a + b
print("Сума чисел =", suma)

Ті самі приклади, але через функції з параметрами

1. Привітання (через функцію з параметром)

Функція greet(name) приймає ім’я користувача як аргумент і виводить привітання. 


def greet(name):
    print("Привіт,", name + "!")

name = input("Введіть своє ім'я: ")
# Виклик функції
greet(name)

2. Сума двох чисел (через функцію з параметрами)

Функція sum_two_numbers(a, b) приймає два числа і повертає їх суму. 


def sum_two_numbers(a, b):
    return a + b
    
a = int(input("Введіть перше число: "))
b = int(input("Введіть друге число: "))

# Виклик функції
suma = sum_two_numbers(a, b)
print("Сума чисел =", suma)

Алгоритми з розгалуженням (умовний оператор)

Без функцій

1. Перевірка на повноліття

Користувач вводить свій вік. Якщо йому 18 років або більше, програма повідомляє, що він повнолітній, інакше – що він ще неповнолітній. 


age = int(input("Введіть свій вік: "))

if age >= 18:
    print("Ви повнолітній")
else:
    print("Ви ще неповнолітній")

Через функції

1. Перевірка на повноліття (через функцію)

Функція check_age(age) приймає вік користувача і виводить результат перевірки. 


def check_age(age):
    if age >= 18:
        print("Ви повнолітній")
    else:
        print("Ви ще неповнолітній")

age = int(input("Введіть свій вік: "))
# Виклик функції
check_age(age)
