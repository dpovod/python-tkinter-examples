Можна обрати свою тему, але структура має бути схожа.

План роботи

  1. Оберіть тему сайту
  2. Продумайте структуру
  3. Знайдіть текст і зображення
  4. Створіть HTML-сторінки
  5. Зробіть меню на кожній сторінці
  6. Перевірте, чи всі сторінки пов’язані
  7. Додайте стилі (CSS)

Чекліст

  • ✅ У мене є 3–5 сторінок
  • ✅ Кожна сторінка має назву (не "Сторінка 1")
  • ✅ Є меню на кожній сторінці
  • ✅ Всі сторінки пов’язані між собою
  • ✅ Є текст і зображення
  • ✅ З головної сторінки можна перейти всюди

Правила

  • ❌ Не можна робити "Сторінка 1 / 2 / 3"
  • ❌ Не можна робити сайт без меню
  • ❌ Не можна робити сторінки без тексту
  • ❌ Сторінки повинні бути пов’язані між собою

Базовий HTML-шаблон


<!doctype html>
<html lang="uk">
  <head>
    <meta charset="UTF-8" />
    <title>Мій сайт</title>
    <link rel="stylesheet" href="style.css" />
  </head>

  <body>
    <nav>
      <a href="index.html">Головна</a>
      <a href="events.html">Події</a>
      <a href="contacts.html">Контакти</a>
    </nav>

    <div class="container">
      <h1>Заголовок</h1>
      <p>Текст...</p>
    </div>

    <div class="container">
      <h1>Ще один блок</h1>
      <p>Ще текст...</p>
    </div>
  </body>
</html>

Базовий CSS


/* Базові налаштування */
body {
    font-family: Arial, sans-serif;
    margin: 0;
    background-color: #f5f5f5;
    color: #333;
    font-family: Arial, sans-serif;
    line-height: 1.6;
}

/* Контейнер */
.container {
    max-width: 900px;
    margin: 0 auto;
    padding: 20px;
    background: white;
    overflow: hidden;
}

/* Заголовки */
h1,
h2,
h3 {
    color: #222;
}

/* Меню */
nav {
    background-color: #333;
    padding: 10px;
    margin-bottom: 20px;
}

nav a {
    color: white;
    margin-right: 15px;
    text-decoration: none;
    font-weight: bold;
}

nav a:hover {
    text-decoration: underline;
}

/* Зображення */
img {
    max-width: 100%;
    height: auto;
    display: block;
    border-radius: 8px;
}

/* Списки */
ul {
    padding-left: 20px;
}

/* Картки (можна використати) */
.card {
    border: 1px solid #ddd;
    padding: 15px;
    margin-top: 15px;
    border-radius: 8px;
    background: #fafafa;
}
