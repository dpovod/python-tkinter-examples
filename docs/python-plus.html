⭐ Додаткові задачі з Python (підвищений рівень)

📌 У всіх задачах цієї сторінки дані повинні вводитися з клавіатури (за допомогою input()).

Під час налагодження дозволяється тимчасово використовувати захардкоджені значення.

1. Паліндром

Користувач вводить слово. Перевірити, чи є воно паліндромом.

2. Друге найбільше число

Користувач вводить числа через пробіл. Знайти друге найбільше число.

Заборонено використовувати sort().

3. Перевірка впорядкованості

Користувач вводить список чисел через пробіл. Перевірити, чи впорядкований він у зростаючому порядку.

4. Циклічний зсув

Користувач вводить числа через пробіл. Виконати циклічний зсув вправо на один елемент. 


Приклад:
1 2 3 4
Результат:
4 1 2 3

5. Видалення дублікатів

Користувач вводить числа через пробіл. Створити новий список без повторюваних значень.

Порядок елементів зберегти.

6. Найчастіше число

Користувач вводить числа через пробіл. Знайти число, яке зустрічається найчастіше.

Якщо таких кілька — можна вивести будь-яке.

7. Міні-калькулятор (меню)

Створити програму з меню:


1 – додавання
2 – віднімання
3 – множення
4 – ділення
0 – вихід

Користувач вводить номер дії та два числа. Програма працює у циклі, поки не буде введено 0.

8. Підрахунок статистики

Користувач вводить числа через пробіл. Вивести:

  • мінімум
  • максимум
  • середнє арифметичне
  • кількість парних чисел

9. Зсув на k позицій

Користувач вводить числа через пробіл та число k. Виконати циклічний зсув вправо на k позицій.

10. Перевірка на однакові елементи

Користувач вводить числа через пробіл. Перевірити, чи всі елементи списку однакові.
