Відображення геометричних фігур за домогою модуля Canvas

Приклад коду для створення полотна з прямокутником за допомогою Canvas:


from tkinter import *  # Імпортуємо все з модуля tkinter

window = Tk()  # Створюємо головне вікно

# Створюємо полотно (Canvas) розміром 200x200 пікселів з білим фоном
c = Canvas(window, width=200, height=200, bg='white')
c.pack()  # Розміщуємо полотно у вікні

# Малюємо прямокутник
c.create_rectangle(
    60, 80, 140, 190,   # координати діагоналі
    fill='yellow',      # колір заповнення
    outline='blue',     # колір межі
    width=3             # ширина межі
)

window.mainloop()  # Запускаємо головний цикл обробки подій
