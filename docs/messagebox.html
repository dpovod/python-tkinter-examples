Вікно з кнопкою, при натисканні на яку відображається вікно повідомлення (messagebox)

Ось приклад коду для створення вікна повідомлення за допомогою Tkinter:


import tkinter
import tkinter.messagebox

# Функція для відображення вікна повідомлення
def show_message():
    tkinter.messagebox.showinfo("Відповідь", "Hello Again!")

# Створення головного вікна
window = tkinter.Tk()
window.title("Hello World!")

# Текст на вікні
label = tkinter.Label(window, text="Hello World!")
label.pack(pady=10)

# Кнопка
button = tkinter.Button(window, text="Привіт", command=show_message)
button.pack(pady=10)

# Запуск програми
window.mainloop()
