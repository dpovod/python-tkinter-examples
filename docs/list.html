Задачі на роботу зі списками (list)

📌 У задачах №1–11 списки (масиви) потрібно придумати самостійно та захардкодити в коді програми (не вводити з клавіатури).

🔎 Нагадування: функції та методи


# функції викликаються окремо та приймають список як аргумент
numbers = [1, 2, 3]
len(numbers)
max(numbers)

# методи викликаються через крапку після змінної
numbers.append(4)
numbers.remove(2)
numbers.reverse()
numbers.copy()

👉 Функція працює зі списком.
👉 Метод належить самому списку і викликається через крапку.

1. Створення різних списків

Створіть програму, у якій задано списки, що складаються з:
1) назв уроків у вашому розкладі в один обраний день тижня;
2) оцінок, які ви отримали протягом навчання у 8 класі з інформатики;
3) даних, що описують: ваше ім’я, зріст і повну кількість років.

2. Сума елементів списку

Дано список чисел. Знайти суму всіх елементів списку.
Підказка: можна використати функцію sum() або цикл for..in з накопичувачем.

3. Кількість парних чисел

Дано список цілих чисел. Порахувати, скільки серед них парних.
Підказка: for..in + if + оператор % (остача від ділення).

4. Найбільше число у списку

Дано список чисел. Знайти найбільший елемент.
Підказка: можна використати функцію max() або цикл for з порівнянням.

5. Найменше число у списку

Дано список чисел. Знайти найменший елемент.
Підказка: можна використати функцію min() або цикл for.

6. Середнє арифметичне

Дано список чисел. Знайти середнє арифметичне всіх елементів.
Підказка: sum(list) / len(list) або цикл з підрахунком суми та кількості.

7. Реверс списку

Дано список. Створити новий список у зворотному порядку.
Підказка: можна використати методи copy() + reverse(), зріз [::-1] або цикл.

Приклад:
[1, 2, 3, 4] → [4, 3, 2, 1]

8. Фільтрація додатних чисел

Дано список чисел. Створити новий список тільки з додатних чисел.
Підказка: можна використати for + if, filter() або list comprehension.

9. Кількість входжень числа

Дано список чисел і число x. Порахувати, скільки разів число x зустрічається у списку.
Підказка: можна використати метод count() або цикл for.

10. Подвоєння елементів

Дано список чисел. Створити новий список, де кожне число помножене на 2.
Підказка: цикл for, list comprehension або map().

Приклад:
[1, 3, 5] → [2, 6, 10]

11. Видалення нулів

Дано список чисел. Створити новий список без нульових значень.
Підказка: for + if, filter() або list comprehension.

Приклад:
[0, 1, 0, 3, 5, 0] → [1, 3, 5]

Операції зі списками

12. Конкатенація списків

Дано два списки чисел. Створити один спільний список, що містить елементи з обох списків.
Підказка: можна використати оператор + або метод extend().

13. Додавання елемента в список

Дано список чисел. Додати нове число в кінець списку.
Підказка: можна використати метод append().

14. Вставка елемента за індексом

Дано список чисел. Вставити число x у список за вказаним індексом.
Підказка: можна використати метод insert(index, value).

15. Видалення елемента за значенням

Дано список чисел. Видалити зі списку число x.
Підказка: можна використати метод remove() або цикл for.

16. Видалення елемента за індексом

Дано список. Видалити елемент за заданим індексом.
Підказка: можна використати pop() або оператор del.

17. Фільтрація списку

Дано список чисел. Створити новий список тільки з чисел, що більші за 10.
Підказка: можна використати цикл for + if або list comprehension.

18. Пошук довжини списку

Дано список. Знайти кількість елементів у списку.
Підказка: можна використати функцію len() або цикл for..in з лічильником.

19. Перевірка наявності елемента

Дано список і число x. Перевірити, чи є число x у списку.
Підказка: можна використати оператор in або цикл for.

20. Копіювання списку

Дано список. Створити його копію.
Підказка: можна використати метод copy(), зріз [:] або цикл.
